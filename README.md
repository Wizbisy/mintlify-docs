# Aegis Documentation

This repository contains the documentation for Aegis.

## Local Development

To run the documentation server locally and preview your changes, use `npx`:

```bash
npx mintlify dev --port 3001
```

Open your browser to `http://localhost:3001` to view the live preview.

## Deployment

The documentation is automatically deployed via the Mintlify GitHub app whenever changes are pushed to the main branch.

## Getting Help

For support, please contact the Aegis core team.
