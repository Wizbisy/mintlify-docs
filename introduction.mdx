Welcome to the Aegis API documentation.

Aegis is a secure financial infrastructure layer built for **autonomous AI Agents**. Through a single API, developers can provision programmable wallets for their agents, enforce strict spending policies, and enable agents to autonomously discover and pay for external API services using the **x402 Micropayment** protocol on the **Arc Network**.

This documentation explains how Aegis works, what you can build with it, and how to get started, from your first API call to live autonomous payments.

### What is Aegis?

Aegis is the financial backbone for AI agent economies. It gives your agents the ability to hold, spend, and manage money safely.

With Aegis, your AI Agents can:

- Receive a secure **Developer Controlled Wallet** on the Arc Network (powered by Circle)
- Hold and manage balances in **USDC**
- Discover paid API services through the **x402 Marketplace**
- Autonomously pay for external APIs using the **x402 Micropayment** protocol
- Transfer USDC to any EVM address on Arc
- Bridge USDC across chains via **Circle CCTP**
- Operate under strict **spending policies** enforced at the database level
- Register autonomous **limit orders** and **DCA schedules** executed by the Wealth Sentinel
- Allocate across multiple yield protocols with **multi-yield deposits** (Aegis Vault + Synthra V3)

## Core Features

### Agent Onboarding

Register agents via a secure email OTP flow. Each agent receives a unique API token and a provisioned wallet, no private keys to manage.

### x402 Marketplace

Search the Circle Discovery API for paid services, inspect pricing before committing, and execute payments, all through simple REST calls.

### Financial Actions

Transfer USDC, bridge across chains, swap tokens, and pay for services. Every action is audited, idempotent, and policy checked.

### Spending Policies

Set per transaction maximums, daily caps, and monthly budgets. The policy engine rejects any action that would exceed the configured limits.

### Wealth Engine

Register limit orders and DCA schedules that execute autonomously via the Wealth Sentinel. Allocate deposits across the Aegis aUSDC Vault and Synthra V3 concentrated liquidity positions for optimized yield generation.
